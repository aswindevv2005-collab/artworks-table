# Artworks Data Table

This project displays artwork data from the Art Institute of Chicago API
using PrimeReact DataTable.

The application implements server-side pagination and persistent row selection.

---

## Features

- Fetches artwork data page by page from the API
- Server-side pagination
- Checkbox-based row selection
- Selection persists across page navigation
- Custom row selection for the current page only
- No prefetching of other pages

---

## Tech Stack

- React
- TypeScript
- Vite
- PrimeReact
